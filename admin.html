🔒 Trang quản trị nội dung

Trang này chỉnh nội dung website (dữ liệu ở js/data.js) ngay trên trình duyệt. Đây không phải hệ thống backend — GitHub Pages là hosting tĩnh nên không thể tự lưu file. Sau khi chỉnh xong, bạn xuất file data.js mới rồi thay thế + commit lên GitHub.

Xuất file data.js

Bấm “Tạo nội dung data.js”, sau đó tải file xuống hoặc copy nội dung, ghi đè vào js/data.js trong repo GitHub của bạn rồi commit + push. Trang web (index.html) sẽ tự động dùng nội dung mới ngay khi file được publish.
